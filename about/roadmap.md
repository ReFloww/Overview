# Roadmap

ReFlow development milestones and future plans.

## Phase 1: Smart Contract Development & Audit

- Core smart contract architecture design
- LoanToken ERC-20 implementation
- Factory pattern development (FactoryP2P, FactoryManager)
- Investment and repayment logic
- Secondary market functionality
- Comprehensive security audit

## Phase 2: MVP Testnet

- Deployment to Mantle Sepolia testnet
- Backend API development (NestJS)
- Frontend application (Next.js)
- Wallet integration and Web3 connectivity
- End-to-end testing
- Bug fixes and optimizations

## Phase 3: Integration with Restock.id & Launch

- Full integration with Restock.id lending platform
- KYC/AML compliance implementation
- Mainnet deployment on Mantle Network
- Production infrastructure setup
- Public launch
- User onboarding and support

## Phase 4: Multichain Expansion

- Expand to additional EVM-compatible chains
- Cross-chain asset support
- Bridge integrations
- Broader market accessibility

## Phase 5: Institutional API Integration

- Enterprise-grade API development
- Institutional investor onboarding
- Advanced analytics and reporting
- White-label solutions for partners
- B2B integration tools
