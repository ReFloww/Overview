# Restock.id Partnership

ReFlow operates in strategic partnership with Restock.id, a licensed P2P lending platform in Indonesia. This partnership is fundamental to how ReFlow works.

## About Restock.id

Restock.id is an established peer-to-peer lending platform that:

- Is licensed and regulated by OJK (Indonesia's Financial Services Authority)
- Specializes in lending to productive sectors
- Has an established track record in loan origination
- Maintains comprehensive borrower vetting processes
- Handles all collection and default management

## Role of Restock.id

### Loan Origination

Restock.id handles all borrower-facing activities:

| Function | Description |
|----------|-------------|
| **Applications** | Receives and processes loan requests |
| **Vetting** | Verifies borrower identity and business |
| **Assessment** | Evaluates creditworthiness |
| **Pricing** | Sets interest rates based on risk |
| **Disbursement** | Funds loans to borrowers |

### Credit Assessment

Thorough evaluation includes:

```
Credit Assessment Process
├── Business Analysis
│   ├── Financial statements
│   ├── Cash flow projections
│   └── Business viability
├── Borrower Verification
│   ├── Identity check
│   ├── Business registration
│   └── Track record
├── Collateral Evaluation
│   ├── Asset verification
│   ├── Valuation
│   └── Legal documentation
└── Risk Grading
    ├── Score assignment
    └── Grade (A-E)
```

### Loan Servicing

Throughout the loan term:

| Activity | Description |
|----------|-------------|
| **Payment Collection** | Receives borrower repayments |
| **Monitoring** | Tracks loan performance |
| **Communication** | Manages borrower relations |
| **Reporting** | Provides performance data |

### Default Management

When borrowers have difficulty:

| Stage | Action |
|-------|--------|
| **Early Warning** | Proactive outreach |
| **Late Payment** | Collection procedures |
| **Restructuring** | Workout arrangements if viable |
| **Default** | Collateral liquidation |
| **Recovery** | Distribution to investors |


## Loan Flow

### Complete Journey

```
1. BORROWER APPLIES (Restock.id)
   Business submits loan application

2. ASSESSMENT (Restock.id)
   Credit evaluation, collateral verification

3. APPROVAL (Restock.id)
   Loan terms set, documentation completed

4. TOKENIZATION (ReFlow)
   Loan converted to tokens, listed on platform

5. FUNDING (Investors via ReFlow)
   Investors purchase loan tokens

6. DISBURSEMENT (Restock.id)
   Funds released to borrower

7. SERVICING (Restock.id)
   Payment collection, monitoring

8. DISTRIBUTION (ReFlow)
   Token holders receive repayments

9. COMPLETION
   Loan paid off, tokens burned
```
