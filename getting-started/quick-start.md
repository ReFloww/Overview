# Quick Start Guide

Get started with ReFlow in just a few steps. This guide will have you investing in minutes.

## Prerequisites

Before you begin, you'll need:

- A Web3 wallet (MetaMask, Rainbow, or WalletConnect-compatible)
- MNT tokens for gas fees (on Mantle Network)
- USDT for investments
- Valid ID for KYC verification

## Step 1: Set Up Your Wallet

### Add Mantle Network to Your Wallet

| Setting | Value |
|---------|-------|
| Network Name | Mantle Sepolia (Testnet) |
| Chain ID | 5003 |
| RPC URL | https://rpc.sepolia.mantle.xyz |
| Currency Symbol | MNT |
| Block Explorer | https://sepolia.mantlescan.xyz |

### Get Test Tokens (Testnet Only)

1. Visit the [Mantle Sepolia Faucet](https://faucet.sepolia.mantle.xyz/)
2. Connect your wallet
3. Request test MNT tokens

## Step 2: Create Your Account

1. Visit the ReFlow platform
2. Click **"Connect Wallet"**
3. Select your wallet provider
4. Approve the connection request

## Step 3: Complete KYC Verification

1. Navigate to **Profile** â†’ **Verification**
2. Provide required information:
   - Full legal name
   - Email address
   - Phone number
   - ID document
3. Submit for verification
4. Wait for approval (usually within 24 hours)

## Step 4: Choose Your Investment Mode

### Option A: Managed Mode (Recommended for Beginners)

1. Go to **Auto-Manage** section
2. Browse available fund managers
3. Review their performance history
4. Select a fund and investment amount
5. Confirm your investment

### Option B: Self-Managed Mode

1. Go to **Market** section
2. Browse available loan products
3. Review loan details:
   - Interest rate
   - Duration
   - Risk grade
   - Borrower sector
4. Click **"Invest"**
5. Enter amount and confirm

## Step 5: Track Your Portfolio

1. Visit **Dashboard** for overview
2. Check **Portfolio** for detailed holdings
3. Monitor **Transaction History** for all activities
