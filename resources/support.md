## Getting Help

### Before Contacting Support

Please gather the following information:

1. **Your wallet address** (public address only, NEVER share private key)
2. **Transaction hash** (if applicable)
3. **Screenshot** of the issue
4. **Browser and wallet** you're using
5. **Steps to reproduce** the problem

### What to Include in Your Request

```
Subject: [Brief description of issue]

Wallet Address: 0x...
Transaction Hash: 0x... (if applicable)
Browser: Chrome/Firefox/Safari
Wallet: MetaMask/WalletConnect/Other

Description:
[Detailed description of the issue]

Steps to reproduce:
1. [First step]
2. [Second step]
3. [What happened vs what was expected]

Screenshots attached: Yes/No
```

## Bug Reports

### Reporting a Bug

1. Check if issue is already reported
2. Gather reproduction steps
3. Submit via GitHub Issues or email

### Bug Report Template

```
**Description:**
[Clear description of the bug]

**Expected behavior:**
[What should happen]

**Actual behavior:**
[What actually happens]

**Steps to reproduce:**
1. [Step 1]
2. [Step 2]
3. [Step 3]

**Environment:**
- Browser: [e.g., Chrome 120]
- Wallet: [e.g., MetaMask 11.0]
- OS: [e.g., Windows 11]

**Screenshots:**
[Attach if applicable]
```

## Bug Bounty Program

We reward responsible disclosure of security vulnerabilities.

### Scope

- Smart contracts
- Web application
- API endpoints
- Authentication systems

### Rewards

| Severity | Reward |
|----------|--------|
| Critical | $5,000 - $20,000 |
| High | $2,000 - $5,000 |
| Medium | $500 - $2,000 |
| Low | $100 - $500 |

### Rules

- Do not exploit vulnerabilities
- Do not access other users' data
- Report privately before disclosure
- Provide clear reproduction steps

Submit to: security@reflow.io
