## Technical Requirements

### Infrastructure

- Cloud hosting (AWS, GCP, or Azure)
- PostgreSQL database
- Redis cache
- CDN for static assets

### Blockchain

- Mantle Network RPC access
- Deployer wallet with MNT
- Contract verification setup

### Integrations

- KYC/AML provider
- Email service (SendGrid, AWS SES)
- Analytics (optional)
