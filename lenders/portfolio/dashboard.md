# Dashboard Overview

The Dashboard is your at-a-glance view of portfolio performance, recent activity, and key metrics. This guide explains each component.

## Dashboard Layout

```
┌─────────────────────────────────────────────────┐
│  Portfolio Summary Cards                         │
├─────────────────────────────────────────────────┤
│  Performance Chart          │  Allocation Chart │
├─────────────────────────────────────────────────┤
│  Recent Activity            │  Upcoming Events  │
└─────────────────────────────────────────────────┘
```

## Portfolio Summary Cards

### Total Portfolio Value

- Combined value of all investments
- Updates in real-time
- Includes principal + accrued interest

### Invested Capital

- Total amount you've invested
- Excludes returns
- Baseline for ROI calculations

### Total Returns

- All earnings to date
- Includes distributed and accrued
- Shown as absolute and percentage

### Active Investments

- Count of current positions
- Links to full portfolio view

## Performance Chart

### Time Ranges

View performance across different periods:
- 7 Days
- 30 Days
- 90 Days
- 1 Year
- All Time

### What It Shows

- Portfolio value over time
- Return rate trends
- Comparison to benchmark (optional)

### Reading the Chart

| Trend | Meaning |
|-------|---------|
| **Upward slope** | Growing value (returns accumulating) |
| **Flat sections** | Stable period (waiting for repayments) |
| **Dips** | Withdrawals or realized losses |
| **Spikes** | Large repayments received |

## Recent Activity

Shows latest transactions:

| Type | Description |
|------|-------------|
| **Investment** | New positions opened |
| **Repayment** | Principal/interest received |
| **Trade** | Secondary market activity |
| **Withdrawal** | Funds removed |

Each entry shows:
- Date and time
- Transaction type
- Amount
- Asset involved
- Status

## Upcoming Events

Calendar of expected events:

- **Upcoming Repayments**: Scheduled distributions
- **Loan Maturities**: Positions ending soon
- **Fund Distributions**: Managed mode payouts

## Metrics Deep Dive

### Average APY

Weighted average return across all holdings:

```
Average APY = Σ(Position Value × APY) / Total Portfolio Value
```

### Time to Maturity

Average days until positions mature:
- Shorter = More liquidity
- Longer = Locked longer

### Default Exposure

Percentage of portfolio in troubled loans:
- 0% = Healthy portfolio
- > 5% = Review allocations
