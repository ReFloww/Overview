# Integration Guide

Integrate ReFlow into your platform or application.

## Integration Options

| Option | Best For | Complexity |
|--------|----------|------------|
| API Integration | Custom applications | Medium |
| Webhooks | Event-driven systems | Low |
| White-Label | Full platform deployment | High |
| Partner SDK | Quick integration | Low |

## Use Cases

### For P2P Lending Platforms

- Offer blockchain-based liquidity to lenders
- Enable secondary market trading
- Provide transparent portfolio management

### For Financial Applications

- Integrate loan tokenization
- Add DeFi capabilities
- Offer new investment products

### For Fund Managers

- Access to curated loan portfolios
- Automated investment management
- Performance tracking tools
