# Understanding Your Portfolio

Your portfolio is your command center for tracking investments, monitoring performance, and managing your P2P lending activities on ReFlow.

## Portfolio Overview

The Portfolio section provides a comprehensive view of all your investments across both managed and self-managed modes.

## Key Metrics

### Portfolio Summary

| Metric | Description |
|--------|-------------|
| **Total Value** | Current value of all holdings |
| **Invested Capital** | Original amount invested |
| **Total Returns** | Earnings to date |
| **Unrealized Gains** | Profit not yet collected |
| **Active Positions** | Number of current investments |

### Performance Metrics

| Metric | Description |
|--------|-------------|
| **Portfolio APY** | Overall annualized return |
| **This Month** | Current month's performance |
| **All-Time Return** | Cumulative return percentage |
| **Projected Earnings** | Expected future returns |

## Portfolio Components

### Active Investments

Displays all current loan positions:

```
Active Investments
├── Managed Mode Holdings
│   ├── Fund Manager A
│   └── Fund Manager B
└── Self-Managed Holdings
    ├── Loan Token 1
    ├── Loan Token 2
    └── Loan Token N
```

### Pending Transactions

Shows transactions in progress:
- Pending investments
- Pending withdrawals
- Pending trades

### Completed

Historical investments:
- Matured loans
- Sold positions
- Fully repaid investments

## Navigating Your Portfolio

### Dashboard Tab
Quick overview of key metrics and recent activity.

### Holdings Tab
Detailed list of all current investments with:
- Investment amount
- Current value
- Return rate
- Days remaining
- Actions available

### Performance Tab
Charts and analytics:
- Returns over time
- Comparison to benchmarks
- Grade distribution
- Sector breakdown

### History Tab
Complete transaction history with filters.

## Understanding Holdings

Each holding shows:

| Field | Meaning |
|-------|---------|
| **Token/Fund** | Investment identifier |
| **Principal** | Original investment |
| **Current Value** | Present worth |
| **Accrued Interest** | Earned but not distributed |
| **APY** | Annual percentage yield |
| **Maturity** | Expected end date |
| **Status** | Active, Pending, Completed |

## Portfolio Actions

### From Portfolio Page

| Action | Description |
|--------|-------------|
| **View Details** | Full investment information |
| **Add More** | Increase position |
| **Withdraw** | Remove funds (managed mode) |
| **Sell** | List on secondary market |
| **Track** | Set alerts and notifications |

## Portfolio Health

### Diversification Score

Your portfolio receives a diversification score based on:
- Number of positions
- Spread across risk grades
- Sector distribution
- Duration variety

### Risk Analysis

| Risk Level | Criteria |
|------------|----------|
| **Low** | 80%+ in Grade A/B, well-diversified |
| **Medium** | Balanced grade distribution |
| **High** | Concentrated or Grade D/E heavy |
